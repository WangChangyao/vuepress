初识vuepress (此网站基于vuepress开发)

vuepress基于vue开发、config.js配置、markdown语法的一套快速构建静态网站的框架；

目录结构解析 (docs模式)

.vuepress-->config.js 配置页面主题,全局可使用$site获取此配置对象，方便自定义主题开发
docs目录下:为路由文件

    README.md-------------------路由path为/,执行此markdown
    文件名/README.md------------路由path为此文件名，执行此文件名下的markdown

工作原理

在config.js文件中配置全局对象参数，在每个页面可通过$site获取此对象参数，页面ui可随意设计，形成了主题可自定义开发；
在vue页面中，使用router-link可进入docs目录下配置好的路由；
makedown语法和vue结合开发页面；

搭建个人主题

1.创建主题模块：在.vuppress目录下创建layouts文件夹，在layouts下创建Layout.vue;创建成功，vuepress会立即使用Layout.vue作为首页入口，之前使用的主题即失效；

创建styles目录，创建index.styl,vuepress会自动使用此文件进行样式渲染

2.修改config.js

3.开发导航栏、左边标题栏和内容栏

Layout.vue

    <div class="page">
                /*顶部导航栏*/
               <div class="topBar">
               /*$site.title为config.js中配置的title */
                   <p class="navTitle">{{$site.title}}</p>
                   <span class="line">|</span>
                   /*$site.description为config.js中配置的description*/
                   <p class="navDescripe">{{$site.description}}</p>
                   /*tabs 为config.js中themeConfig中获取，$site.themeConfig.tabs*/
                   <tab-nav style="flex: 1" :tabs="tabs"></tab-nav>
               </div>
               <div class="navsBox">
                    /*左侧侧标题栏*/
                   <div class="navs">
                        /*tabs 为config.js中themeConfig中获取，$site.themeConfig.navs*/
                       <nav-bar :navs="navs"></nav-bar>
                   </div>
                   /*内容栏*/
                   <div style="flex: 1;height: 100%;overflow-y: scroll;">
                       <div style="width: 800px;margin: 0 auto">
                        /*不同路由挂在不同的内容*/
                           <Content slot-key="slidecontent"></Content>
                       </div>
                   </div>
               </div>
           </div>

tab-nav导航栏组件

<template>
    <div class="box">
        <div class="item" v-for="tab in tabs" :key="tab.link">
            <a target="_blank" :href="tab.link">
                <span>{{tab.text}}</span>
                <img v-if="tab.icon" :src="tab.icon" alt="">
            </a>
        </div>
    </div>
</template>

<script>
    export default {
        props:['tabs']
    }
</script>

nav-bar标题栏组件

<template>
    <div class="nav">
        <div class="item" v-for="(nav,index) in navs" :key="nav.link">
            <div v-for="(routes,key) in nav" :key="key">
                <p>{{key}}</p>
                /* 导航进config.js中navs配置的路由link*/
                <router-link v-for="route in routes" :to="route.link" :class="$route.path == route.link? 'active':''">{{`${route.text}`}}</router-link>
            </div>
        </div>
    </div>
</template>

<script>
    export default {
       props:["navs"]
    }
</script>
